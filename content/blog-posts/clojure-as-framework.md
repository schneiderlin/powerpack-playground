表达力强的语言对框架的需求就少, 因为框架的主要作用就是把各种相关的库组合起来. 表达能力强/组合能力强的语言, 本身就可以作为框架.
clojure 的生态很强, 可以通过 interop java 吃到 JVM 的生态.
还可以通过 clojurescript 吃到 JS 和 node 的生态. 
libpython-clj 可以吃到 python 的生态
